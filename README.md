# Portfolio Personal Web

## 📋 Descriere

Portfolio personal web interactiv și responsiv care prezintă proiectele, abilitățile tehnice și informațiile de contact. Aplicația este dezvoltată cu tehnologii web moderne și oferă o experiență optimă pe toate dispozitivele.

## 🚀 Funcționalități

- Design responsive pentru desktop, tabletă și mobil
- Secțiune prezentare personală cu foto și bio
- Galerie interactivă de proiecte cu filtrare
- Secțiune abilități tehnice cu grafice progress bar
- Formular de contact funcțional
- Animații smooth scroll și hover effects
- Mod dark/light theme

## 🛠️ Tehnologii Utilizate

- **HTML5** - structura semantică a paginii
- **CSS3** - stilizare avansată, flexbox, grid, animații
- **JavaScript (Vanilla)** - interactivitate și funcționalitate
- **Font Awesome** - iconițe
- **Google Fonts** - tipografie

## 🔧 Instalare și Rulare

### Cerințe preliminare
- Browser web modern (Chrome, Firefox, Safari, Edge)
- Editor de cod (VS Code recomandat)

### Pași instalare

1. Clonați repository-ul:
```bash
git clone https://github.com/username/portfolio-personal.git
cd portfolio-personal
```

2. Deschideți `src/index.html` în browser:
```bash
# Opțiune 1: Deschidere directă
open src/index.html  # macOS
start src/index.html # Windows

# Opțiune 2: Folosind Live Server în VS Code
# Instalați extensia Live Server
# Click dreapta pe index.html → Open with Live Server
```

## 📱 Utilizare

1. Navigați prin secțiunile folosind meniul principal
2. Explorați proiectele în galeria interactivă
3. Vizualizați abilitățile tehnice în secțiunea Skills
4. Contactați prin formularul dedicat
